# Expresiones canarias

| Palabra / expresión              | Significado en castellano estándar                                 | Ejemplo                                              |
| -------------------------------- | ------------------------------------------------------------------ | ---------------------------------------------------- |
| Chacho / chacha                  | Tío/tía, oye, muchacho/a                                           | "Chacho, ven pa'cá."                                 |
| Ños                              | Interjección de sorpresa, dolor o impresión                        | "Ños, qué calor."                                    |
| Agüita                           | Admiración o exageración                                           | "Agüita, qué pasada."                                |
| Fleje                            | Mucho, gran cantidad                                               | "Había fleje de gente."                              |
| Pelete                           | Frío intenso                                                       | "Anoche hacía un pelete..."                          |
| Calufa                           | Mucho calor                                                        | "Qué calufa hace hoy."                               |
| Jilorio                          | Hambre, sensación de vacío en el estómago                          | "Tengo un jilorio..."                                |
| Magua                            | Pena, lástima, nostalgia                                           | "Me dio magua no ir."                                |
| Guagua                           | Autobús                                                            | "Voy en guagua."                                     |
| Cholas                           | Chanclas, sandalias cómodas                                        | "Ponte las cholas."                                  |
| Papas                            | Patatas                                                            | "Papas arrugadas con mojo."                          |
| Millo                            | Maíz                                                               | "Piña de millo."                                     |
| Cotufas                          | Palomitas de maíz                                                  | "Compramos cotufas para la peli."                    |
| Barraquito                       | Café típico de Tenerife                                            | "Me pedí un barraquito."                             |
| Guachinche                       | Local típico de comida casera y vino del país                      | "Vamos a un guachinche."                             |
| Mojo                             | Salsa canaria, roja o verde                                        | "Con mojo picón."                                    |
| Gofio                            | Harina tostada de cereales                                         | "Escaldón de gofio."                                 |
| Baifo                            | Cabrito; en expresión, despiste                                    | "Se me fue el baifo."                                |
| Se me fue el baifo               | Me despisté / metí la pata / se me olvidó                          | "Iba a decir algo, pero se me fue el baifo."         |
| Machango                         | Payaso, muñeco, persona ridícula                                   | "No seas machango."                                  |
| Arretranco                       | Trasto inútil; persona torpe o inútil                              | "Quita ese arretranco de ahí."                       |
| Tolete                           | Tonto, torpe                                                       | "Estás hecho un tolete."                             |
| Mataperrería                     | Travesura, gamberrada                                              | "Deja las mataperrerías."                            |
| Fos                              | Qué asco / mal olor                                                | "Fos, eso huele fatal."                              |
| Enyesque                         | Aperitivo, picoteo                                                 | "Pon algo de enyesque."                              |
| Tenderete                        | Fiesta, reunión con comida/música                                  | "Se montó un tenderete."                             |
| Belingo                          | Fiesta, juerga                                                     | "Están de belingo."                                  |
| Palique                          | Conversación larga, charla                                         | "Nos echamos un palique."                            |
| Golifiar                         | Cotillear, curiosear                                               | "Está golifiando."                                   |
| Novelero/a                       | Cotilla, curioso, metomentodo                                      | "No seas novelero."                                  |
| Puntal                           | Persona valiosa, crack, alguien de confianza                       | "Eres un puntal."                                    |
| Mi niño / mi niña                | Forma cariñosa de dirigirse a alguien                              | "Mi niño, ¿qué te pongo?"                            |
| Fisquito / fisco                 | Un poco, una pequeña cantidad                                      | "Dame un fisquito."                                  |
| Chinijo/a                        | Niño pequeño                                                       | "Los chinijos están jugando."                        |
| Alegar                           | Hablar, conversar                                                  | "Nos pusimos a alegar."                              |
| Alongarse                        | Asomarse                                                           | "No te alongues tanto."                              |
| Cambado/a                        | Torcido, doblado                                                   | "Eso está cambado."                                  |
| Pachorra                         | Calma excesiva, lentitud                                           | "Qué pachorra tienes."                               |
| Peguijera                        | Insistencia pesada, lata                                           | "Qué peguijera con eso."                             |
| Relajo                           | Desorden, cachondeo, falta de seriedad                             | "Esto es un relajo."                                 |
| Godo/a                           | Peninsular; puede sonar despectivo según tono                      | "Ese acento es de godo."                             |
| Chicharrero/a                    | Persona de Santa Cruz de Tenerife; a veces, de Tenerife en general | "Es chicharrero."                                    |
| Ustedes                          | Uso habitual en lugar de "vosotros"                                | "¿Ustedes vienen?"                                   |
| Más nunca                        | Nunca más                                                          | "No voy más nunca."                                  |
| No cojas lucha                   | No te agobies, no te preocupes                                     | "Déjalo, no cojas lucha."                            |
| Le falta un agua / un chaparrón  | Le falta madurar / le falta un hervor                              | "A ese le falta un agua."                            |
| Arrancadilla                     | Última copa antes de irse                                          | "Nos tomamos la arrancadilla."                       |
| Irse pa'l choso                  | Irse a casa                                                        | "Me voy pa'l choso."                                 |
| Ya el conejo me enriscó la perra | Se me fastidiaron los planes                                       | "Con esta avería, ya el conejo me enriscó la perra." |
